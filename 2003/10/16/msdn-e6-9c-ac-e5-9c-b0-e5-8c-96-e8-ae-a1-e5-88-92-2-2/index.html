MSDN本地化计划

建立我们共同的激情社区–中国开发者网络 致MVP的一封信

 

各位MVP和技术专家们

 

你们好！

 

最近我们将针对MSDN 进行一个全面和大的升级计划，主要是会增设许多新的栏目，将更多的英文MSDN网站进行本地化，增加具有深度的技术文章、专栏、.NET Show等等。 我们已经制定了具体的计划，来实现这个目标，其中有一个非常重要的环节是我们将采用和英文版同步的方法来更快的获得最新的信息，以方便下一步MSDN中文版也支持RSS/XML和查找的功能。

 

所以我们新设置了一个Developer Center ，这里将有许多主题和这些主题相关的文档，我们计划MSDN中文版将包括下面的主题区：

1.      .NET Framework (new feature)

2.      Architecture ( would Focus on PAG guides ) (new feature)

3.      ASP.NET   (new feature)

4.      Security   (existing currently)

5.      Visual Basic    (new feature)

6.      Visual C#        (new feature)

7.      Visual C++      (new feature)

8.      Visual J# (new feature)

9.      Visual Studio (new feature)

10.  Web Services        (new feature)

11.  SQL Server     (new feature)

12.  Mobile Web Development           (new feature)

13.  Pocket PC & Smartphone            (new feature)

14.  SharePoint Products and Technologies     (new feature)

15.  Table PC                       (new feature)

16.  DirectX                                 (new feature)

17.  Embedded Operating System Development    (new feature)

18.  .NET Compact Framework                  (new feature)

19.  .NET Security                              (new feature)

20.  Building Distributed Applications with .NET      (new feature)

21.  XML                                      (new feature)

22.  Mobility and Embedded                      (new feature)

23.  Office                                          (new feature)

24.  Windows Developer             (existing currently)

25.  Data Access Technologies    (existing currently)

26.  Component Development    (existing currently)

27.  Software Development Policy     (existing currently)

28.  Web Development        (existing currently)

29.  Network & Communication (existing currently)

30.  Exchange                     (existing currently)

31.  Install & Systemer Management (existing currently)

32.  Graphics                (existing currently)

33.  User Interface                     (existing currently)

 

由于栏目变得非常多，所以对于英文原文的选取上存在一个高质有效的原则，所以我们邀请您帮助我们，大家一起建立一个大家喜欢看，认为是最价值的中文版的MSDN—中国开发者网络。 我想各位都是不同领域的专家和技术热爱者，那么现在请你花一点时间，将你认为这你的领域中最有价值的MSDN的英文文章的意见告诉我们。 “我认为这文章非常Cool，对开发人员非常有价值，它应该有中文版本！”

“我认为这篇文章相当不错，解决了工作和项目的问题，应该让更多的人知道”

“这是非常有见解和启发我思维的文章，我想应该有更多的知道，并且开发人员可以针对这个进行讨论”

。。。。。。。。

 

告诉我们你认为最有价值，或对你和开发人员最有帮忙的MSDN文章。我们将需要你的支持、你的帮助和你参与，让我们一起来建立这个新的中文版的MSDN—中国开发者网络。

 

具体的你可以将你选的文章注明它的分类(上面的33个分类)并且附文章题目和URL 写在Email中发Email给qcheng@microsoft.com ，Email标题包含“MSDN SCL”这样的字母。截至时间是10月27日，这期间任何时间您都可以将您的想法告诉我们。

 

另外我们也新增加了: MSDN Magazine, Columns 和.NET Show的栏目

 

To add the localized MSDN Magazine,

including the in-depth technical articles for the professional and senior developers who are interested in Microsoft products and technologies.

 

To add the localized topics of **Columns**

a.      Adventures in Visual Basic .NET

b.      At Your Service

c.       Code Secure

d.      Driving DirectX

e.      Office Talk

f.        Wonders of Windows Forms

g.      Working With C#

Adding the 9 columns of MSDN Magazine, we will have 16 localized columns, it will exult the professional and senior developers in China.

 

MSDN TV and .NET Show

1.       MSDN TV

  1. The .NET Show

 

杂志、技术专栏和.NET Show 这三部分如果你也有好的建议，也请按上面的方法的给我们建议。

 

 

这是非常好的一次机遇和挑战，我们希望通过这个计划能更大程度的提高我们获得新技术的速度以及和开发人员的联结，提高开发人员的学习效率和舒适度，同时中国开发者网络也将成为即英语、日语和法语之后，第四大语系的MSDN本地社区 ，迈向MSDN Worldwide最高排名Best of Best的行列。而这一切都需要您的支持、您的参与和激情。
