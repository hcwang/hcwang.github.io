几个镜头

1.小气的神的主页今天改版了，有一些金属质感，阅读起来好多了。不知道大家怎么看？看来小气的神同学的确是一位极好的美工，下次让他咱们也改改版，呵呵。我这儿的Logo还一直没有人提供呢:(，另外，名字也需要改一下，可是也没有好名字。

2.开发者俱乐部越来越没落了，上个世界就在Chinaasp混,已经很有感情了，虽然现在改到htttp://www.dev-club.com去了，可是仍然割舍不掉，今天去看，同时在线才一百多人，十分感叹！估计再此下去，会沦落为下一个Chinaasp。

3.昨天收到银行的帐单，上月话费为七百多元，其中五百元整的GPRS费用。轰轰烈烈的GPRS维权案到此结束，八月份按二百元算的，九月份就是五百元，十月份我再也没有用GPRS了。过几天我准备去关掉GPRS服务。

4.我快没有钱了，没有工作，没有收入来源，Best又不让我找工作。过几天我得向大家要求捐款了，希望大家到时候还认识我，一百元以下的票子我不要的，要多给一些呀。就当我结婚演习吧，呵呵。收一下份钱:D

超级尿布清洗机转转同学提供的MVP资源使用经验

由于最近垃圾邮件泛滥，很多兄弟没有收到微软发出的一些重要邮件。我受Grace委托，在此简要说明这封邮件的内容并且共享一下我的经验/教训。其中黑色部分是邮件原文，红色部分是我的说明。

mall> support@mvpaward.com

各位一定要特别注意来自于这个信箱地址的邮件，通常是一些非常重要的信息。在此我建议各位尽量使用国外的邮箱（最好是hotmail的，毕竟是微软自己家的么），否则丢信的情况可能会比较严重。

mall> zzwww@yahoo.com

尊敬的 Wei Zhang：

Microsoft 很高兴地通知您，由于您在与全球技术社区中表现出卓越的乐于助人的精神，我们特授予您为 Microsoft Most Valuable Professional (MVP)的称号。

祝贺您！同时也感谢您过去一年来对 Microsoft 技术社区中其他用户的大力帮助。Microsoft 高兴地向您赠送一个 MVP 奖励礼包，作为我们对您作出的突出贡献表达谢意的特殊方式。

如果您还没有收到奖励礼包，不要着急，相信很快就会收到（下面提供奖励包跟踪信息）。您的奖励包包括以下内容：

 

§ 一份“感谢”礼物。

§ 一张认可您为 Microsoft? Most Valuable Professional

的获奖证书。

§ 两份守密协议 (NDA)，协议书和回复信封。

§ 关于其它权益的信息，包括访问 Microsoft t Company Store 及 MSDN 或 订阅TechNet。

这一部分就是很多兄弟比较关心的大礼包。里面好东东还是不少的。如果还没有收到，请见后面的内容。

要了解有关“MVP 计划”的详细信息，请访问：

http://mvp.support.microsoft.com/。

建议各位有时间的时候去这个站点看看，有些问题就迎刃而解了……

** 重要事项： 

 

此电子邮件包含有关如何获取更多权益的信息。

以前的获奖者如果登录 MVP AWARD SITE 查看 NDA 并激活其它权益，就会发现今年的权益履行措施发生了很大变化。

请仔细阅读此电子邮件，并对其信息保密。

MVP AWARDS SITE – 梦想开始的地方

=============================

 

要获取其它 MVP 权益，请访问 MVP AWARD SITE：

www.mvpaward.com

 

您需要用自己的 MVP ID 和密码登录。  MVP ID

是您参与“MVP 计划”的个人标识符。  请在与“MVP

计划”所进行的关于您 MVP 权益的任何通信中附上此ID号码。

 

   您的 MVP ID 是：  1234   

 

您登录 MVP AWARD SITE 的密码是：  12345678。

在奖励网站中，您可以在线查看 NDA（“守密协议”）。  接受在线 NDA 可以为您激活长达 60 天的权益，届时 Microsoft 需要收到您的签名书面文件，才能确保将权益的激活时间延长至一整年。

上面这一部分是关于MVP AWARD SITE的说明，这个站点及其重要。我们需要在此进行注册以获得一些MVP资源（如MSDN的订阅）。注册时使用上面的用户名/密码就可以了。每个MVP的都不相同，所以各位要尽量找到此邮件，否则注册将成问题。 我要特别强调的是，这个注册是一次性的。注册后信息就不能修改了。所以注册时一定要准备好自己的相关信息（如英文地址），并且要一次完成。我对此有深刻教训，由于我中断了注册过程，导致MSDN的邮寄地址为空…… 在正确的完成MVP AWARD SITE的注册过程之后。将会收到 support@mvpaward.com发出的MSDN订约确认信件：

mall> support@mvpaward.com | ????? | ????????? 

???mall> zzwww@yahoo.com

 可以看到，由于我的失误，导致邮寄地址错误……所以各位在完成注册的时候一定要谨慎啊……

在确认MSDN的订阅之后，可能会等待一段时间（这段时间的长短不一定，看个人的地理位置和英文地址的正确性了。如果时间长的话，各位也不要太着急。因为MSDN是从你收到第一个邮包的时候开始计算的，拿到的越晚，其中包括的内容反而会更多）。收到一封MSDN的订阅信件。然后再按照信中的提示完成MSDN的注册，不久你就会收到一个巨大的纸箱（如果选择CD介质的话）……此后一年中的每个月你都应该收到一个邮包，其中是MSDN更新的内容。

o 要了解有关 MSDN Universal 的详细信息，请访问： 

http://msdn.microsoft.com/subscriptions/prodinfo/levels.asp

o 要了解有关 TechNet Plus 的详细信息，请访问：

http://www.microsoft.com/technet/treeview/default.asp?url=/technet/subscriptions/wanttn.asp

? 访问 Microsoft Company Store – Redmond, WA

o 提供“库存”软件（美式英语）、硬件（NTSC，美式英语）、书籍和促销物品。

o 邮寄费用和关税由 Microsoft 支付。

? 访问 MVP Private Newsgroup（私有新闻组）和 MVP 专有网站。

o 访问 Beta 版产品。

o 访问 MVP Academy 以了解在线学习计划。

o 通过 MVP Wish 对 Microsoft 产品和“MVP 计划”提出更改建议。

上面这些是我们可以在可以使用的一些资源。比如我们可以选择订阅一年的MSDN Universal 或 TechNet Plus。而Microsoft Company Store则是微软的电子商店，我们有150$的电子货币，可以在其中以很低廉的价格购买各种商品（好象这一届MVP可以购买商品的种类要比前几届的多，各位可不要看花眼哦）。而商品的邮寄费用和关税都是由微软支付的，我们可以不必为此担心（这个比微软自己的员工还要优惠，他们是要自己支付这一部分的费用的）。建议各位不要象开心就好那样一上来就花光光，这样将来有好东西的时候再后悔就晚了……呵呵;）

电子商店的地址是https://shop.ecompanystore.com/microsoft/MIC_Login.asp。注册的时候在这个页面上点击MVP按钮，然后输入自己的MVP编号和last name（MVP编号应该在support@mvpaward.com发出的另一封邮件之中）。然后在完成后面的注册过程即可。

再次提醒各位一定要准备好自己正确的英文地址，否则在购买商品和MSDN邮寄的时候都会遇到麻烦的。
