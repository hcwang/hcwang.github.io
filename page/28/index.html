19岁西华大学的学生生让微软服了！！

发表于 2003-10-17 | 分类于 未分类

转贴自：http://pop.pcpop.com/ShowHomeRecommend.aspx?Recommendid=471

刚入校一个月，西华大学软件工程和网络技术学院大一学生梁振便成了学校名人：他9月底被微软公司评为“最有价值专家”(Most Valuable Professional，简称MVP)，同时受邀参加明年在美国微软总部举行的全球MVP峰会。据悉，梁振也是成都获此殊荣的第一人。
　　据了解，微软“最有价值专家”称号是微软的一个年度奖项，专门授予那些在论坛中交流微软相关技术表现出色的专家，在技术人群中极具影响力。目前全球共有MVP1600多名，中国目前仅150余名。
“建筑师”改行学软件
　　19岁的梁振是地道的成都娃，从没接受过系统学习的他完全靠自学而成为计算机平台技术高手。从高中起便开始自创网站并做网站维护。
　　该校软件工程和网络技术学院副院长黄学彬有几分得意地向记者透露，梁振这个“专家”是他们好不容易“挖”到该学院的。父母都从事建筑工作的梁振在升本考试中原本要报考建筑专业，软件学院得知梁振在网络技术上的特殊才能，便由副院长亲自出马游说梁振进入该学院学习。
自办网站当“活雷锋”
　　同学们告诉记者，梁振不仅是义务电脑维修工，而且热心与人分享他的专业技术，是活脱脱一个“活雷锋”。
　　输入网址www.diyinside.cnred.net，梁振告诉记者这是他和4位爱好者一同自筹资金创建的公益网站，免费为需要帮助的人提供技术服务、文章和讲座。“我们不为赚钱”，记者了解到梁振曾拒绝了微软公司和个人的赞助。梁振的精湛技术和这种热情助人精神成为他当上MVP的重要原因。


随笔及文章FAQ

发表于 2003-10-17 | 分类于 未分类

阅读本篇之前请先阅读《新手指南》。

1.如何发布图片？

如果您的图片已经存在于互联网上，那么使用编辑器下方的“Design”与“HTML”切换按钮将设计视图切换为HTML视图，然后在该状态下进行贴图。如果您的图片没有位于互联网上，可以先使用“相册”功能将其上载于本服务器上，再按照上面的方法如法泡制。

2.编辑器支持从页面上直接拷贝粘贴吗？

是的，编辑器支持从其它页面拷贝超文本内容直接粘贴，但提请大家注意，一定要在发表后重新审查一下，拷贝过来的内容是否破坏了本站首页及个人首页的“排版”，以1024*768为准，否则系统管理员有权在警告无效后对其进行删除或者修改。

新手指南

发表于 2003-10-17 | 分类于 未分类

本指南会根据实际情况进行更新，希望大家进行查看。

在大家获得登录帐号后，请根据邮件中的登录帐号来到您的页面，比如您的帐号为joy，那么您的页面将为http://blog.joycode.com/joy，使用您的用户名及密码登录后，在导航中将出现第四个链接，即“管理”，点击它进入管理界面。

在管理界面中，初始页是“随笔”的管理。另外，我们在上面的Tab里面，还可以看到“文章”，很多朋友可能对“随笔”及“文章”的区别搞不太明白，我们在这儿做一个简单介绍：

  • “随笔”中的贴子将会出现在首页中，而“文章”中的贴子将只出现在“文章”的分类中，所以在您增加文章类型的贴子时，一定切记要把它加入到一个分类中，否则您的朋友将无法浏览到您的文章。
  • 就博客文化来说，“随笔”中的贴子更加随意一些，可以写三言两语或者生活感悟之类，但“文章”中的贴子基本上都很正式，适合珍藏，所以您正式的作品请尽量放在“文章”中。
  • 就我们来说，在“随笔”中我们鼓励原创，所以希望凡是转贴的东西请尽量加到“文章”中。使本站更有原创气氛。
  • “随笔”及“文章”中的内容均支持MSN Messenger图标，大家可以参考http://messenger.china.msn.com/Resource/Emoticons.aspx网址中的图标定义。不过在评论中尚不支持，请大家注意。

“反馈”是一个十分有用的界面，在此界面中，我们可以看到所有发给您的评论信息，您可以对其进行删除等管理操作。

“链接”界面用以管理您的“友情链接”，并且可以将其分门别类的进行管理。另外，很多朋友可能想加一些Logo类型的链接，其实可以在链接文字中输入HTML代码的，这样，我们就拥有了Logo链接了。当然，我们并不鼓励您这样做，因为Blog界面最好尽量简单一些。另外，链接中的RSS地址只针对对方的地址是博客类型的网站而定义的，您可以在其界面上发现有一XML或者RDF的图标，它所指向的地址就是RSS地址，RSS是博客站点上广泛应用的一种协议，用以进行新闻聚合之类的工作，比如http://www.flashanywhere.net/mxna就是一个极好的聚合新闻站点。

“相册”用以上传您的图片，目前，我们对图片上传没有任何限制，但请大家不要上传过大的图片，另外，图片也要放在分类中，否则外界仍然访问不了。您的图片会自动被系统分为三种尺寸的，不需要您手工裁剪了，以利于显示。如果您需要在您的贴子中出现图片，建议选择尺寸较小一些的。提醒大家，节约空间。同时，有人可能想上传文件，十分对不起，目前本系统不具备这种功能。但在不远的将来，我们可以为大家提供此种功能。请等待。

“统计”是一个非常有用的功能，可以查看您哪些贴子更受欢迎，目前“检查页访问量”功能暂时不可用，希望在.Text下一版本的时候，这个功能会出现。而访问纪录则可以使用，可以查看各种访问者的信息，如果您点击访问信息后面的“创建”按钮，则会创建一个“引用”在您的贴子反馈中。（很多人不知道为什么自己的首页上的“统计”中的“引用”始终为零，其实这就是关键）。

最后就是“选项”了，选项中共有四项，分别介绍如下：

  • “配置”：用以配置自己页面的基本信息，比如更改帐号、标题及描述等信息都在此处。我发现很多新加入的博客，往往不更改自己的标题及描述信息，希望能够看到此信息后，尽快更改。另外，界面也是在此处更改的，我们将随着时间逐渐增加新的界面，希望大家留意。同时，这里面的邮件也十分有用，建议大家都留真实邮件，有两个好处：一，别人可以通过您首页上的“联系”与您通信;二，如果有人看了您的文章发表了评论，可以即时通过邮件系统发送给您。当然，这儿的电子邮件不会显示给访客的。
  • “关键词”：这也是非常有用的功能，往往我们在文章里面需要写一个单词，希望能够自动被系统替换，比如，我希望每次输入开心就好的时候，就会自动给我替换为开心就好的页面的链接，这样，您就可以在此处进行定义。
  • “密码”：这就不用说了，是用来更改您的密码的。
  • “性能”：这里面的三个选项是用来微调性能的。

最后，需要指出，虽然本站点是在.Text 0.94系统上构建的，但是我们已经进行了很多额外的工作，并且仍然在继续完善中，欢迎大家给我们提供新的点子。
