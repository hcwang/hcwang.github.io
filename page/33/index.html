陪我九年的香烟

发表于 2003-10-12 | 分类于 未分类

从高中开始，开始向酷看齐，学会了抽烟，而后，它就一直陪伴我左右。寂寞时，它陪我;难过时，它陪我;开心时，它陪我;写程序时，它陪我;高谈阔论时，它陪我。无论我何种心态，它对我总是不离不弃。

也曾经想过与它恩断情绝，彻底斩断对它的情思。可是每当看到别人手中的香烟时，总会让我感觉有一丝丝冲动，拿起烟来的冲动，所以，一直到现在，它仍然在陪着我走那段人生的路。

迷茫的时候，我在它的帮助下，找到了方向，找到了目标。困惑的时候，它带给我灵感，让我发现柳暗花明。每次当我想戒烟的时候，总会找到成千上万的理由阻止我这样做。

在山东读的高中，那时候抽的牌子是“大鸡”，这是山东最出名也最普及的牌子。无论是人力车夫，还是高级白领，山东人似乎对“大鸡”情有独衷，甚至当我在北京读书的时候，从山东来的哥们给我捎的礼物也是一条“大鸡”香烟。

大一的时候，摆阔，开始抽“希尔顿”，甚至有时候也会抽一盒“万宝路”。大二的时候，烟瘾开始上来了，抽不起这种高档烟，开始抽带薄荷味的“威龙”，后来发现了一种北京更加流行的牌子：“都宝”，挺便宜，但却陪我最久，前几天，在回通州的出租车上，还有一位出租车司机给我一支“都宝”，让我回忆起大学那段迷茫的时光。

记得大三还是大四的时候，北京出来一个自有品牌“中南海”，跑到我们学校门口做宣传，看男生经过时，会塞给两支装的“中南海”一包。我那时候以为他们是卖烟的，冲着他们微微笑，摆摆手，说我不抽烟，可是走过之后，才知道是赠品，遂跑回去向人家索要。人家同样微微笑，摆摆手，说你不是不抽烟吗？还好，刘大明白要了一盒，送给了我，从此，我迷上了“中南海”。

中南海很柔，很轻，总觉得这样对身体损害最小，所以五年来，我一直在抽中南海，而且是那种10毫克的。当我的工资上了五位数的时候，曾经以为自己不再穷了，想换换口味，于是选了5毫克的那种，觉得更“派”一些。可是抽了不到一个月，觉得经济上承受不起，于是算了。

两次真正的戒烟经历，一次是毅力戒烟，似乎没有坚持一周，就半途终止了，因为实在受不了那种袅袅升起的烟雾的诱惑。去年，在父母的支持下，决定药物戒烟。选了一种喷药，每当想抽烟的时候，就往咽喉处喷一下。可是我后来发现喷了那种烟后，抽烟的感觉更爽。这样不仅要买烟，还要买喷烟，于是，这次戒烟也以失败告终。

也知道另外一种很灵的戒烟药，只需要贴在穴道上即可，之后，再闻到烟味就会恶心。可是IT圈的朋友们相聚，哪能没有一个不抽烟的？总不能在饭桌上有人点起了香烟，我就呕吐吧？所以只是心向往之，从未买过此药物。

陪我九年的香烟，你还要陪我多少年？

第二轮面试

发表于 2003-10-10 | 分类于 未分类

今天上午的面试是由一位部门大区经理来负责的，面试的气氛热烈而坦诚。

Playyuer及其它MVP一直在问我，都考了啥，我想都受了很多外面关于面试考题的影响，所以一直认为，考官应该问我井盖为什么是圆的之类的问题，其实呢，面试的很简单，连我都没有想到。

今天的问题比昨天的多了一些，大约花了将近半个小时吧，还是跟昨天问的差不多，最经典的一个问题是：你认为做售前最重要的三个素质。我回答如下：1.说服力，要让客户接受我们推销的产品及技术。2.激情，这是微软的文化，一定要体现出来。3.诚实，但是要有一定的技巧。考官对我的回答十分满意，尤其是最后一点我附加的那句。

然后我们开始了坦诚的对话，他们需要的是一个对微软的产品线非常熟悉的人，也就是知识面要全，而且最好有销售经验等等。其实我在第一天就注意到这个问题了。而且这个职位出差非常严重，负责北方十三省市，随时要候命，还要做各种渠道及内部培训。

最后，我们讨论了一些MVP资源的利用问题，他让我组织一些北京的MVP出来转转，然后邀他一起。他们以后有一些项目可以外包给我们做。 (Grace应该给我奖赏）。

面试结束后，我又去了王静茹女士的Developer Team，先办妥我的银行转帐问题。然后找上次在TechED遇到的两位朋友讨论一下他们将MVPs组织起来做一些事情的计划。并且留下了通讯方式。

MVP是很好的资源，需要我们自己发动起来，做一些事情。大家有什么好的意向吗？

收到MVP大礼包

发表于 2003-10-10 | 分类于 未分类

今天收到了MVP大礼包，据DHL的人告诉我说，这个礼包10月2日就到海关了，可见速度很快（有MVP问，从美国寄东西到中国来多久，现在好象只2-3天就可以了）。

礼包里面有以下物件：

  • 背包，非常漂亮的，瑞士产背包，有瑞士军刀的标志，绣着MVP的标志，而且背着也比其它包舒服。
  • MVP证书，一个罗利·摩尔以及Sean（应该是，我没有仔细看）签字的MVP证书，可以挂在墙上，也可以像相框一样摆在桌子上，非常不错。
  • NDA：装在一个彩色信封里面，里面还有很多其它的东西，比如一些介绍MVP计划的彩页等等。
  • MVP胸饰：这东西容易被大家忽略，一定要注意，在那个彩色包装箱里面，粘在内壁上，很好玩的玩艺，可以挂在西装或者T恤上。不是很引人注目的那种。

感谢微软又给一个包 ，这包应该是我见到的最好的微软的包裹了:P

通过第一轮面试了

发表于 2003-10-09 | 分类于 未分类

今天早上七点半起床，然后上网查一下邮件，并且使用Windows Update升级一下我的操作系统，然后一直到九点左右，从家里面出发到微软公司。相信很多参加TechED的朋友去参观过微软的新办公大楼了，我面试的地方与我九月份在那儿工作的楼层一样，十九层。

在十点左右，到达微软公司，然后在前台传了一下部门秘书出来迎接我（我手里面还有一张微软的门禁卡，现在仍然有效，九月份办的）。在前台后面的那个大会议室等待面试我的老兄。

其实我心里面一直不紧张，说心里话，对这个职位并不是很期待，因为这是一个Sales的工作，更认真的说，是一个售前的工作。而且是V打头的（通过微软员工的邮件地址，我们可以知道一些附加信息，如i-xxx@microsoft.com代表此人是intern性质的，v-xxx@microsoft.com代表此人是Vendor性质的，所以我们在这儿要恭喜Grace MM现在已经是正式职工了，因为她的邮箱改了，对吧？)

面试过程不到二十分钟，就问对微软哪些产品熟悉之类的问题，没有想像中那样玄，不过呢，估计是因为第一轮的原因。

面试结束后，我用门禁卡进入微软里面，找我几个朋友，在楼下的咖啡店里面喝了一小时咖啡，然后又再叫上一个朋友，中午一起吃饭。

下午继续在微软，并且办理一下我上次工作的报酬支付问题，需要办一张中国银行的卡（见到了久仰的王静茹女士）。然后跟Best商量如何花我的$150美金，竟然发现微软员工每人竟然有一千美金的虚拟货币，比我们多多了，哈哈，于是我们俩疯狂物，AP一次买两，室内篮球买三，再加上一些包等等林林总总花了三四百美金，可是支付的时候出现了问题，只能寄往美国，而且还要出钱，所以我们后来取消了。

在下午14:41左右，收到了面试成功的电话，让我明天上午十点进入第二轮面试。

看来要过五关，斩六将了。
